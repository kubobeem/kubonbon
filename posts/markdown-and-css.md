# Markdownで書いてCSSで魅せる

このブログは、記事内容はMarkdownで管理して、見た目は共通CSSで整える構成です。

## 書き方はシンプル

- 見出しは `#` で書く
- 箇条書きは `-` で書く
- 強調は `**太字**` で書く

## コードブロック

```ts
const message = "Markdownで書けば公開まで速い";
console.log(message);
```

> 文章に集中できるのがMarkdown運用の強みです。
